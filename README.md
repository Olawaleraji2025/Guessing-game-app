# Guess the Number Game 

Hey there! Welcome to **Guess the Number**, a fun and challenging web-based game that puts your intuition to the test.

## Features 

- **Multiple Difficulty Levels**: Choose from Easy (1-50), Medium (1-100), or Hard (1-200) to match your mood and skill level.
- **Lives System**: You've got limited guesses depending on difficulty - use them wisely!
- **Smart Hints**: The game gives you helpful feedback like "Too high!" or "Way too low!" to guide you.
- **Guess History**: See all your previous guesses at a glance, so you can learn from your attempts.
- **High Score Tracking**: Your best scores are saved locally, so you can keep pushing for that personal record.
- **Mystery Reveal**: The secret number stays hidden until you guess it or run out of lives.
- **Responsive Design**: Plays great on desktop, tablet, or mobile devices.


## Technologies used

- **HTML5**: Structures the game interface and content.
- **CSS3**: Handles all the styling, animations, and responsive design.
- **JavaScript**: Powers the game logic, scoring, and local storage.
- **Local Storage**: Remembers your high scores between sessions.
